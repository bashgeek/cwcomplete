CwComplete - Javascript Quick Autocompleter

This is a small mootools autocompletion tool

For more information: Homepage/Documentation/Feedback »

Works..

  • ..in Safari 3+, Firefox 3+, IE 6, IE 7, IE 8, Chrome and Opera 9+
  • Requires mootools 1.23+ and (for example) PHP to deliver a list of values

Features..

  • Easy to setup for most cases
  • Mouse and keyboard selection (up/down, escape, enter) possible
